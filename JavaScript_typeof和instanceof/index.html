typeof和instanceof

2017-01-02

基本类型

基本类型有以下五种：

  1. Undefined
  2. Null
  3. Boolean
  4. Number
  5. String

    typeof操作符

    用于检测变量的数据类型
    检测结果有以下6种：
  6. “undefined” 未定义
  7. “boolean” 布尔类型
  8. “string” 字符串
  9. “number” 数字
  10. “object” 对象或null
  11. “function” 函数
